Solar Checkup

Free tool to check your Enphase solar panels for underperformance, degradation, and failing microinverters.

No data collected Runs in your browser Open source
1
Get your panel data from the Enphase Gateway

Your Enphase IQ Gateway has a local API that reports per-panel production data. You'll open it in your browser, copy the data, and paste it here. Takes about a minute.

1
Open your gateway in a browser tab: https://YOUR_GATEWAY_IP
Find the IP in your router's device list, or in the Enphase app under System → Network. If you get a security warning, click Advanced → Proceed (the gateway uses a self-signed certificate — this is normal).
2
Log in to the gateway. If it asks for a token, you'll need to generate one:
  1. Go to entrez.enphaseenergy.com/entrez_tokens
  2. Log in with your Enphase account
  3. Enter your Gateway serial number (on the device label or in the Enphase app)
  4. Select "owner" as the role
  5. Click Generate and copy the token
  6. Paste it on the gateway login page and click Submit

Tokens last about 12 months. If you previously logged in through "Login with Enphase", you may already be authenticated.

3
Once logged in, navigate to the inverter data URL in that same tab:
https://YOUR_GATEWAY_IP/api/v1/production/inverters
You should see a page of JSON data — text with square brackets and numbers. This is your per-panel production data.
4
Select all the text on that page (Ctrl+A / ⌘A), copy it (Ctrl+C / ⌘C), and paste it below.
This data contains only microinverter serial numbers and wattage readings — no personal information.
